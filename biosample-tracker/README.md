# Biosample Tracker

Simple instructions so anyone in the group can run the project.

## What you need

- **Node.js and npm** installed (download from the official Node.js website).

## How to run the project

1. **Open a terminal** (Command Prompt, PowerShell, or any terminal).
2. **Go to the project folder** (where this `README.md` is):

   ```bash
   cd path/to/biosample-tracker
   ```

3. **Install the libraries** (do this once):

   ```bash
   npm install
   ```

4. **Start the app**:

   ```bash
   npm start
   ```

5. **Open the browser** and go to `http://localhost:3000` to see the app.
